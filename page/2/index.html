神的预定和人的责任

发表于 2020-05-20 更新于 2026-02-25 分类于 圣经第一 Disqus：
本文字数： 1.9k 阅读时长 ≈ 2 分钟

路加福音22:22 人子固然要照所预定的去世，但卖人子的人有祸了！

神的预定和人的自由，神的拣选和人的责任，如何能够调和？这是一个时常被煞有其事提出的问题。提问者假定这两者必定存在冲突，然而查考圣经经文，这个假定全无立足之地。圣经并行不悖、非常淡定地将这两者一同摆在台面上，提醒我们需要重新审视何为在神的预定、拣选，何为在人的自由、责任。这一默想首先摒弃人为的预设和“应该”，而以归纳法汇总旧约、新约部分相关经文，并尝试在其基础上得出一个自然而然的结论——如果可以的话。同时，这些经文包括、但不限于“预定”和“责任”这两个话题。

罗马书13:13与奥古斯丁: 时候到了，日期满了

发表于 2020-05-18 更新于 2026-02-25 分类于 圣经第一 Disqus：
本文字数： 1.8k 阅读时长 ≈ 2 分钟

罗马书13:11-14 11 再者，你们晓得现今就是该趁早睡醒的时候，因为我们得救，现今比初信的时候更近了。12 黑夜已深，白昼将近；我们就当脱去暗昧的行为，带上光明的兵器。13 行事为人要端正，好像行在白昼；不可荒宴醉酒，不可好色邪荡，不可争竞嫉妒。14 总要披戴主耶稣基督，不要为肉体安排，去放纵私欲。

十七年前读奥古斯丁忏悔录时，第八卷第十二节所载的悔改经历，让我相当费解，为什么一节看起来“稀松平常”的“道德劝诫”经文（罗马书13:13），会对他产生这么大的冲击，将他的整个生命彻底翻转？现在再读这四节经文，不会再有以前那种无知的想法，可以告白的是：时候到了，日期满了，当罪人的心预备妥当时，神的话就句句带着能力。即便奥古斯丁当时翻到的不是罗马书13:13，而是其他使徒书信中的某节经文，我相信他仍然会在那时悔改更新。

一个只存在于未来的证据

发表于 2020-05-10 更新于 2026-02-25 分类于 圣经第一 Disqus：
本文字数： 661 阅读时长 ≈ 1 分钟

出埃及记4:11-12 摩西对　神说：“我是什么人，竟能去见法老，将以色列人从埃及领出来呢？”　神说：“我必与你同在。你将百姓从埃及领出来之后，你们必在这山上侍奉我；这就是我打发你去的证据。”

“这就是我打发你去的证据”——一个非比寻常的证据，一个把已经放到还未之前、把未来拿到现在的证据。这一证据之所以可能，是因为神应许摩西“我必与你同在”，然而，人需要信心才能领受这证据——因为“信就是所望之事的实底，未见之事的确据”。
